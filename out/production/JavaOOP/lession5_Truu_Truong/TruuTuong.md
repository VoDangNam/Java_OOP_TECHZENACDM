# Trường tượng Abstract class + Interface

## Abstract (0 - 100%) : Phương thức trừu tượng (Không có body)

- chỉ định nghĩa phương thức , còn chuyện body ở phần con sẽ lo
- ẩn các chi tiết xử lí chỉ hiển thị tính năng/kết quả
- Chỉ quan tâm đến kết quả ko quan tâm cách thực hiện

## Interface (100% trừu tượng ko có body hết) sử dụng implement
- Các file trong interface đều là final
